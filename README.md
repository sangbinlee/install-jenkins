# install-jenkins
install-jenkins

  
  1.  Install OpenJDK 17 on Ubuntu 22.04
      - sudo apt update
      - sudo apt install -y openjdk-17-jdk
      - sudo apt install -y openjdk-17-jre
  2. Verify Java JDK Installation
      - java -version
